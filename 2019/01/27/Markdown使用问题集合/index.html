Markdown基本语法

  • 删除线：在加删除线的文字左右分别用两个~~号包起来
  • 斜体：在倾斜的文字左右分别用一个*号包起来
  • 加粗：在加粗的文字左右分别用两个**号包起来
  • 斜体加粗：在倾斜和加粗的文字左右分别用三个***号包起来
  • 分割线：三个或者三个以上的---或者***都可以。

插入图片

  1. 使用图床外链插入图片链接——缺点是图床一但被墙或关闭服务，可能会导致图片全部失效

  2. 使用绝对路径存放图片。eg: C:\picture\1.jpg ——不建议使用，具体原因尚不了解，日后修改

    使用绝对路径时，本地有C盘而服务器端没有C盘，将导致路径错误

  3. 使用相对路径。当图片与.md文档在同一文件夹下时使用——![](图片名字)（注意符号都是英文状态）；当图片在与.md文档同路径的一个文件夹内时使用——![](文件夹名字/图片名字)

显示代码段

实现页内跳转之HTML锚点跳转

  • 在需要跳转到的位置添加锚点，语法如下：
    <span id="jump">跳转到的地方</span>
  • 在需要点击跳转的地方使用上面的id，语法如下：
    [点击跳转](#jump)

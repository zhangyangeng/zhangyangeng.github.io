  • 主要记录了Linux系统下需要熟记的一些命令

基础命令

ls 命令

用来列出所有文件和文件夹

  • ls
  • ls 路径
  • ls 选项 路径
    ls -l 路径：以详细列表显示所有文件
    ls -la 路径：以详细列表显示所有文件和隐藏文件
    ls -lh 路径：以合适的大小显示所有文件

pwd 命令

用来打印当前工作目录

  • pwd

cd 命令

用来改变或切换当前工作目录

  • cd 路径
  • cd ~ 切换到当前用户的家目录

mkdir 命令

用来创建目录

  • mkdir 路径
  • mkdir -p 路径 用来进行多层目录的创建
  • mkdir 路径1 路径2 路径3 用来同时创建多个目录

touch 命令

用来创建文件

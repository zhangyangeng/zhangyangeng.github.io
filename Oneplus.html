一、解锁篇

1.ADB命令法

  • 手机打开开发者模式（设置→关于手机→快速连按5次版本号开启），然后在开发者模式中设置 OEM锁为允许，这个解锁以后就要一直开着，没事不要手欠去关了。并且打开 USB 调试开关。
  • 安装ADB工具包到电脑，ADB下载链接点 这里

  • 启动到 bootloader 模式（设置→开发者选项→打开高级重启），按电源键选择“引导加载器”重启，或者使用adb 命令让手机重启进入 bootloader 界面

  • 首次进入 bootloader 模式下连接电脑要确定驱动自动安装成功，一定要耐心等待并确定驱动安装完成，否则会影响到后面的使用，进 bootloader 刷机可能会出现 waiting for device 的提示

  • 成功进入 bootloader 绿色机器人界面以后，输入以下命令开始解锁

  • 此时手机会出现一个选择界面，通过音量键定位光标到 unlock 的选项，按电源键确认即可解锁了，注意这一步会丢失所有数据，包括照片，所以提前做好备份，不过，解锁终身只要一次，以后的 root 和官方系统升级操作，通常都不会再丢数据。

  • 解锁完成后，通过命令退出bootloader并重启手机

  • 重启手机，自带的应用程序有一部分会丢失，联网后静放一会看是否会恢复回来，如果不行的话手动刷一下当前版本的全量包即可。各版本全量包请点击 这里

  • 全量包刷取方法：下载全量包，放到手机存储的根目录，然后打开手机设置-系统-系统更新-右上角齿轮图标-本地升级

2.工具箱无脑法

  • 这里推荐 @千古风流帝 的工具箱，原文链接在 这里
  • 该工具箱里包含了解锁、TWRP、Root 等一系列无脑操作

3.上锁方法

  • 重新上锁前记得先确定已经完全消除 Root
  • 进入 bootloader 界面，在cmd窗口内输入adb命令

二、REC（即TWRP）篇

1.有 Magisk的方法

  • 下载对应安卓版本的 TWRP卡刷包
  • 在 Magisk刷模块的界面刷入 TWRP即可

2.无 Magisk的方法

  • 下载 这里 的与当前手机系统相对应的文件并且解压文件
  • 手机解锁以后，手机进 fastboot模式，插入数据线
  • 打开以 .bat 结尾的文件 ，输入 fastboot flash boot twrp-boot.img 回车，等待出现两个OK以后就算成功刷入了
  • 然后手机按音量键移动到 recovery mode ，按电源键确认
  • 等手机进 TWRP之后默认是英文，点击 settings，然后点击最右边的小按钮，选择 Chinese 并点击 Set language ，这样就变成中文了
  • 输入密码或者手势进 TWRP主界面，接着点击 高级-一加工具箱-Magisk，安装好后重启系统就可以了
  • 原文看 这里，只看安卓Q的部分

3.一加5t版

  • 下载 Official TWRP APP，选择一加5t，下载最新安卓10的 TWRP 的 img 文件
  • 先用工具箱刷入安卓9的 TWRP
  • 然后在安卓9的 TWRP 中直接刷入之前下载好的安卓10的 img 镜像
  • 然后重启进入恢复模式，即进入新版 TWRP 中
  • 刷入支持安卓10的 Magisk 包（应该在20.1版本以上），不要重启
  • 最后再刷1次新版 TWRP 的 img 镜像即可重启进入系统

  • 在 cmd窗口输入以下命令刷入 Magisk的卡刷包

  • 成功后直接点击重启系统，就能在手机抽屉里找到安装好的 Magisk管理器，ROOT即已完成

  • 注：最新官方版TWRP可以不用以下教程，直接将卡刷包拷贝到手机上刷入

四、系统更新篇

1.同一个安卓版本下的更新

(1).保留 TWRP 的方法

  • 先安装官方的系统更新，建议直接下载全量包然后本地更新，在安装完成之后不要立即去点重启手机

  • 打开 magisk 下载里面搜索 TWRP，选择第一个下载并安装

  • 如果这里无法下载的话，你也可以下载 这里 （提取码: r1b6），然后像刷入模块一样刷入即可

  • 刷入完成后如果想要保留 root 不要重启，返回 magisk 首页执行下面操作

(2).保留 root 的方法（安装最新Magisk）

  • 先安装官方的系统更新，建议直接下载全量包然后本地更新，在安装完成之后不要立即去点重启手机
  • 打开 Magisk Manager，点击安装/更新Magisk，然后选择 安装到未使用的槽位（OTA后） 直到看到“Next reboot will boot to second slot！”，以及提示“All done”，也就是下图右三我红字圈出来的文字
  • 然后重启即可正常成功保留 root

(3).保留 root 的方法（安装旧版Magisk）

  • 提前准备好旧版 Magisk 的卡刷包和 Twrp 的卡刷包
  • 先下载官方的系统更新（或直接下载全量包但不安装），下载完成以后系统会自动安装，此时不要让它安装，立即执行下一步，不然会回到官方 rec
  • 长按关机键，重启到恢复模式，点击安装找到 .ota 文件夹，刷入下载好的系统但不重启
  • 安装 Twrp 卡刷包但不重启系统，返回主界面重启到 rec
  • 安装旧版本的 Magisk 卡刷包并重启

2.不同安卓版本下的更新

(0).保留TWRP和Root ！！！推荐

  • 升级前去 Magisk 取消或卸载所有模块（建议卸载，防止部分模块问题卡开机，可后续再刷入）
  • 将新版的 TWRP（安卓9的 TWRP 不支持安卓10，推荐版本为70或71的 TWRP）和Magisk（不是必须的，但推荐20.1版本）压缩包放到文件根目录，重启手机（这里重启是为了让模块卸载生效）
  • 通过OTA推送或是下载全量包升级，无论何种方式升级，在更新后不要重启手机
  • 到 Magisk 中点击安装，安装到未使用卡槽，安装完后返回，不要重启
  • 进入 Magisk 模块界面，像安装模块一样点击+号，刷入前面放好的 TWRP，不要重启，点击返回
  • 点击 Magisk 右上角，选择重启到 Recovery，重启后等待一会儿，会进入到 TWRP 界面，输入解锁密码进入 TWRP，并刷入之前放好的 Magisk，完成后点击重启，此时，手机已获取永久 Root 权限且保留 TWRP 和 Magisk
  • 手机重启后只需耐心等待一段时间，大概三五分钟，就会进入到 H2OS 界面，开机后输入密码，升级成功，然后可以去 Magisk 刷入其他模块，慢慢享受（建议一个个刷入，防止模块冲突或不适应造成重启卡机）

(1).保留 TWRP 的方法

  • 貌似跨安卓版本时上面的方法并不适用，所以可以试试这种办法
  • 像上面一样安装完系统更新以后，不要重启，进入 Magisk 中刷入与你将要刷的安卓版本相对应的 TWRP
  • 刷完以后不要重启，进行保留 root 的方法

(2).保留 root 的方法

  • 跨版本保留 root 时，尽量先把开启的模块禁用甚至是卸载
  • 然后像上面的方法安装到另一个槽位，然后重启即可

3.有 TWRP 的刷入方法

  • 首先准备需要升的系统的全量包、REC卡刷包和 Magisk面具包

五、官方降级篇

  • 用一加搬家备份手机，备份后所有软件和资料都存在一个名称为 opbackup 的文件夹里
  • 将 opbackup 文件夹复制到电脑里，等以后恢复手机用
  • 在本地升级处刷入专用降级包（即双清全量包）进行降级，降级包下载地址点 这里
  • 开机重启后在本地升级处刷入一加7安卓9的 9.5.8全量包，全量包下载地址点 这里
  • 成功刷回安卓9稳定版，打开一加搬家选择恢复数据
  • 注意：降级并不会回锁，而且如果手机系统应用出现故障，可以通过降级然后直接OTA更新回最新系统

七、9008 救砖篇

0.准备工作

  • 所需工具下载点 这里，密码：lbwa
  • 下载线刷格式的 Rom包，下载下来放到电脑上进行解压
  • 电脑下载安装 Qualcomm 驱动（记得先禁用驱动签名验证，禁用方法点 这里）
  • 这里引用的论坛里的教程，原贴在 这里
  • 注意：如果之前电脑如果有安装过9008驱动的，可能会出现连接手机后一旦把音量键放开，手机就会开机到 fastboot模式，如有出现此情况，你需要连接电脑后按住音量双键不放，并电脑点击桌面计算机右键→设备管理器，找到并卸载干净9008驱动，再重新下载安装「一加7专用的9008驱动」即可

1.安装底层驱动

  • 完全关机状态下按住 左侧音量+ 不放，同时插入数据线，等待 PC 识别到 QHSUSB BULCK设备
  • 打开设备管理器，在 QHSUSB BULCK 上点右键—更新驱动程序—-浏览计算机以查找驱动程序软件，定位到驱动文件夹，点下一步，出现红色警告框选择始终安装此驱动程序，直到提示安装完成
  • 打开设备管理器，查看 端口（COM和LPT），如果出现 Qualcomm HS-USB QDLoader 9008 或者 Qualcomm HS-USB Diagnostics 9006 底层驱动就安装好了

2.开始线刷

  • 把线刷格式的 Rom包电脑上进行解压，解压出来一个文件夹，然后在文件夹里还有一个文件夹，里面就有线刷要用到的文件

  • 进入解压出来的文件夹，接着找到【MsmDownloadTool V3.0.exe】文件，点击鼠标右键以管理员身份运行

  • 接着 完全关机（很重要），即同时按住音量增加键和电源键，在屏幕变黑以后立即松手

    • 还能进入fastboot的：就进入fastboot，按音量键选择power off关掉电源

    • 完全黑砖的：长按电源键40s，等一分钟

  • 同时按住手机的 音量加减键 不放的同时用数据线连接电脑

  • 如果刷机工具没有显示，点击 Enum 键刷新一下

  • 当出现 com端口时表示连接正常

  • 然后点击 Start 开始刷机，刷机过程中软件会有进度条显示，此时就可以松开音量键了

  • 完成后软件会有刷机成功的提示，此时我们拔掉数据线等待开机即可

3.更新系统

  • 线刷仅仅用于救砖，救砖之后还是尽快刷回官方原版的固件系统包
  • 可以 OTA 亦可以直接刷全量包

！注意：

  • 9008 以后只能 OTA 或者卡刷同安卓版本的全量包
  • 以一加5t为例：9008 以后为安卓7.1.1，且该版本无法连接无线网，需要分别卡刷安卓7的最后一个版本、安卓8的第一个版本、安卓8的最后一个版本、安卓9的第一个版本和安卓9的最后一个版本

十、问题集合篇

1.密码问题

  • 手机密码千真万确，但就是提示密码错误进不了系统
  • 方法一：不担心数据，直接9008即可
  • 方法二：担心数据，提取当前系统刷机包的内核，重新刷一下内核即可

2.数据清理问题

  • 双清会导致系统应用丢失，所以需要刷全量包来补救，但又会掉 Root ，所以需要按以下操作
  • 先进 REC进行双清操作，然后刷全量包，刷REC，刷Magisk，刷REC重启。
  • 进系统以后再重启到 REC，格式化 data分区（为了删除内置存储上的文件），此时所有文件被清理，系统应用丢失
  • 然后进系统，用电脑传进手机全量包、REC和Magisk
  • 最后进REC，刷全量包，刷REC，刷Magisk，刷REC重启

3.刷入模块进不了系统问题

  • 按电源+音量减键进入TWRP
  • 在 高级 - 文件管理 - data - adb - modules 删除相应模块即可

4.面具反复刷新问题

  • 如果没有梯子的话，更新面具时就会一直转圈，按下面的方法就可以了
  • 打开面具，在 设置 - 更新通道 - 自定义 中输入以下网址即可：
  • https://qianyegroup.gitee.io/magiskbuilds/updates/custom.json
  • http://magisk.tsyinpin.com/canary.json
  • https://gitee.com/QingFeiDeiYi/Magisk/raw/master/stable.json

5.框架未安装问题

  • 面具成功安装，edxp的模块也成功安装，但是显示未安装
  • 试试将面具设置里面的 核心功能模式 关掉

6.双清、三清、四清

  • 双清：
    wipe data/factory reset：清除用户数据并恢复出厂设置（刷机前必须执行的选项）
    wipe cache partition：清除系统缓存（刷机前执行（系统出问题也可尝试此选项，一般能够解决

  • 三清：
    wipe data/factory reset：清除用户数据并恢复出厂设置（刷机前必须执行的选项）
    wipe cache partition：清除系统缓存（刷机前执行（系统出问题也可尝试此选项，一般能够解决
    Wipe Dalvik Cache：清空虚拟机缓存（可以解决一些程序FC的问题）

  • 四清：
    wipe data/factory reset：清除用户数据并恢复出厂设置（刷机前必须执行的选项）
    wipe cache partition：清除系统缓存（刷机前执行（系统出问题也可尝试此选项，一般能够解决
    Wipe Dalvik Cache：清空虚拟机缓存（可以解决一些程序FC的问题）
    Wipe Battery Stats: 清空电池调试记录数据（刷机前做下这个，会比较好，感觉自己电量有问题的机油也可以试试）

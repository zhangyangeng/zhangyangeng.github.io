介绍

  • 记录在开发微信小程序时的一些知识点

一、知识储备

  • 理解事件机制
  • 理解组件化
  • 理解数据绑定
  • Flex 布局
  • 移动端适配方案

二、基础内容

1.Flex 布局

  • 即弹性布局 —— display: flex;

2.移动端适配

  • 微信底层就进行了 viewport 适配，即 width=device-width

  • 物理像素即为对应的像素点

  • 设备独立像素相当于 CSS 像素

  • dpr：即设备像素比=物理像素/设备独立像素，一般以iPhone 6 的 dpr 为准，即dpr=2。

3.小程序适配方案

  • 在 iPhone 6下 1rpx=1个物理像素=0.5px

4.小程序图片路径

  • 建议使用绝对路径，当绝对路径无法使用时再使用相对路径

5.小程序的文件结构

  1. wxml —— view结构
  2. wxss —— view样式
  3. js —— view行为
  4. json —— 数据及配置

！js中固定格式

  • 注册小程序：App({})
  • 注册页面：Page({})

6.数据绑定

  • 在 data 中初始化页面需要的数据，在页面中可以直接使用

7.事件

  • 冒泡事件：bind+事件名
  • 非冒泡事件：catch+事件名

8.数组与对象

  • 在中括号中的是数组，在大括号中的是对象

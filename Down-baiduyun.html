玩机：手机端破解百度云限速

发表于 2019-02-25 | 分类于 实用教程 | 阅读次数:
字数统计: 821 字 | 阅读时长 ≈ 3 分钟

介绍

  • 该篇介绍了某些通过第三方软件来进行下载百度云文件的方法
  • 该篇所述内容并不一定适用于所有人、所有情况，望知悉
  • 如果有钱的话，当然还是建议入正百度云会员的，毕竟充钱可以解决任何事

利用命令行软件来进行下载

该方法可以解决百度云账号被限速问题。亲测，被限速账号平均下载速度为1M/s，正常账号4M/s。该方法由酷安@谷花泰 提供。查看原文请点击此处

  1. 下载termux软件。直通车：查看链接
    软件刚打开时需要自动下载安装，如果无法安装可能需要翻墙，方法自寻，随便找一个临时免费VPN即可
  2. 授权存储权限，执行完后请允许授权
    执行命令为：termux-setup-storage

  3. 添加软件源
    执行命令为：

  4. 更新软件源
    执行命令为：apt update

  5. 安装BaiduPCS-Go
    执行命令为：apt install baidupcs-go
  6. 开始运行BaiduPCS-Go
    执行命令为：BaiduPCS-Go
  7. 登录百度云账号
    执行命令为：login

  8. 输入账号密码，并验证手机或邮箱

    为了安全起见，密码是不会显示的。如果登录失败，可以输入 login 重新登录

  9. 配置appid
    执行命令为：config set --appid 265486

  10. 配置下载位置，若保存在 /sdcard/百度云 文件下（这里的sdcard指的是手机内存默认目录。请确保你的/sdcard下有这个文件夹，若没有，请自行创建）
    执行命令为：config set --savedir '/sdcard/百度云'
  11. 配置下载线程，官方默认100，可以设置50~500，这里设置为300（理论上线程越大下载速度越快）
    执行命令为：config set --max_parallel 300
  12. 配置并行下载数量，可以设置1~5，这里设置为3
    执行命令为：config set --max_download_load 3
  13. 配置完成后，将termux强行关闭，即清理掉后台。
    打开浏览器，进入百度云官网并登陆账号。
    复制下面网址，在登陆过账号的浏览器中粘贴并访问。
    http://pcs.baidu.com/rest/2.0/pcs/file?app_id=265486&method=list&path=%2F

  14. 访问完后，不管页面显示什么，打开手机版百度云
    在 /我的应用数据 文件夹下会多出一个名为 baidu_shurufa 的文件夹，如果没有，请自建。

  15. 将需要下载的文件复制或移动到 /我的应用数据/baidu_shurufa/ 目录下，比如我这里要下载 动图抓取工具.rar 文件

  16. 打开termux，进入BaiduPCS-Go
    执行命令为：BaiduPCS-Go

  17. 进入 baidu_shurufa 文件夹（若下载多级文件夹下的文件，按此方法递归进入最后一个文件夹即可）
    执行命令为：cd /apps/baidu_shurufa

  18. 下载所需文件即可
    执行命令为：d '动图抓取工具.rar'

  19. 注意，这里下载速度会有浮动，但平均下来也就4M多左右，还是可以接受的。以后下载文件，从第15点开始即可

  20. 如果有什么问题，可以参考作者的文档。直通车：查看链接
